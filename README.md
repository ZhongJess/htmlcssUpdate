# 個人品牌網站 - HTML/CSS 切版練習

六角學院切版專案，實作個人品牌形象網站，包含響應式設計（RWD）。

## 專案結構

```
htmlcssUpdate/
├── index.html          # 主頁面
├── all.css             # 樣式表
├── all.js              # 互動邏輯（jQuery）
├── jquery-3.7.1.min.js
└── img/                # 本地圖片資源
```

## 頁面區塊

- **Header**：Logo + 導覽選單 / 漢堡選單
- **Banner**：主視覺標語
- **Experience**：2B / 2C 工作經驗
- **作品集**：六件作品卡片展示
- **分頁**：頁碼導覽
- **Footer**：聯絡資訊 + 社群連結


## 使用技術

- HTML5 / CSS3
- jQuery 3.7.1
- CSS Flexbox
- RWD Media Queries
